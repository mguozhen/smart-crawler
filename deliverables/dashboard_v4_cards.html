📋 抉择框架

三个问题帮你选

不要看哪个最 sexy，看哪个最匹配你的客户场景。

Q1 · 客户怎么用？
看 vs 用 vs 分析 vs 问 → 决定形态
VIVO 商研每周用 → C Excel Pro
遨森多角色协作 → B Workspace
海尔老板月度看 → A Cockpit
企业 SDR 临时问数 → D Agent Chat
Q2 · 投标场景
demo 现场打动决策者，还是落地深耕？
demo 现场震撼 → A 或 D（最有视觉/科技感）
深耕日常使用 → B 或 C（实用主义）
AI Agent 卖点 → D（这是其他人没有的护城河）
Q3 · 团队能力
能不能 hold 住开发 + 维护？
1-2 人小团队 → C（2 周最快上）
3-4 人 → B（3 周，标准 SaaS）
5+ 人 → A 或 D（更高视觉/技术天花板）
混合方案
不互斥 → B 做主，A 做 demo 入口，D 做 power user 工具
推荐组合：B（主 workspace） + A（投标 demo 大屏） + 渐进式上 D（chat panel 嵌右下角）

不要做 C 单选：C 只能做插件，不能是产品入口
